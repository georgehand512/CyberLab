# Exports Directory

This directory is reserved for versioned VirtualBox appliance exports (.ova) of the CyberLab baseline environment.

Due to file size limitations and software licensing considerations, Virtual Machine appliance files are **not stored in this public repository**.

Baseline exports are maintained offline and versioned according to:

- 07_Snapshot_and_Version_Record.md  
- 06_Change_Log.md  

Example baseline naming convention:
 
- 2026-03-09_CyberLab_v1.0_Baseline.ova  

Each exported baseline corresponds to a documented configuration state and controlled change process.

To recreate the environment, follow:

Documentation/05_Build_and_Configuration_Guide.md
